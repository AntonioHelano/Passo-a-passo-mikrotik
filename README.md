# Passo-a-passo-mikrotik
Auxiliar  na  configuração  do  equipamento 
## Contexto e Objetivos
Este caderno temático foi criado para documentar e organizar o processo de configuração de equipamentos Mikrotik.  
Objetivos:
- Aprender e registrar os principais comandos e configurações.
- Criar um guia prático para futuras consultas.
- Explorar como a IA pode apoiar na organização e revisão do conteúdo.

## Engenharia de Prompts e Cicatrizes
- Prompt inicial: "Explique como configurar DHCP em um Mikrotik."
- Variação: "Liste os passos para configurar DHCP Server no RouterOS."
- Cicatriz: dificuldade em obter exemplos práticos → solução: incluir contexto de rede doméstica no prompt.
- Resultado: respostas mais aplicáveis e detalhadas.

## Miniguia de Estudo

### Resumos Estruturados
- **Configuração inicial**: acesso via Winbox ou WebFig, alteração de senha padrão.
- **DHCP Server**: criação de pool de endereços, configuração da interface.
- **NAT (Masquerade)**: habilitar saída para internet.
- **Firewall básico**: regras de bloqueio e liberação de tráfego.

### Glossário
- **RouterOS**: sistema operacional dos equipamentos Mikrotik.
- **Winbox**: ferramenta gráfica para configuração.
- **DHCP**: protocolo que distribui endereços IP automaticamente.
- **NAT**: tradução de endereços de rede.
- **Masquerade**: técnica de NAT usada para saída de múltiplos hosts.
